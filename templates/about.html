Despre Proiectul "Retete bazate pe ingrediente"

Acest proiect a fost creat de Antal Krisztian si Ghisa Catalin

Proiectul foloseste limbajul Python cu Frameworkul Flask pentru back-end, limbajul HTML cu Frameworkul Zurb Foundation pentru front-end si SQL impreuna cu XAMP pentru crearea si mentinerera bazei de date.
